# FUTURE_ML_01
Machine Learning project to forecast or predict future sales using historical data.
This project focuses on building a sales forecasting model using historical business data.
The goal is to help businesses predict future sales trends and make data-driven decisions.

Problem Statement
Accurate sales forecasting helps organizations:
Plan inventory efficiently
Reduce operational risks
Improve demand planning

Technologies Used
Python
Pandas & NumPy
Matplotlib
Scikit-learn

Output Visualization
The graph below compares actual sales vs predicted sales for the last 30 days.
Close alignment shows good prediction accuracy
Differences highlight sudden demand changes
